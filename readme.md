# Pixel Fishing Project

Минималистичный симулятор рыбалки на C++ и SDL2. Проект разрабатывается двумя участниками в рамках курса по системам управления версиями.

## Разработка
* **IDE:** Visual Studio 2022
* **Язык:** C++17 / 20
* **Библиотеки:** SDL2, SDL_image, SDL_ttf

## Структура проекта
* `src/` — исходный код (.cpp)
* `include/` — заголовочные файлы (.h)
* `assets/` — графика (PNG) и шрифты (TTF)
* `bin/` — скомпилированные исполняемые файлы

## Геймплей и механики
Основной упор сделан на медитативную атмосферу: водная гладь, морской бриз и минималистичный пиксель-арт.

**Логика игры:**
1. **Персонаж:** Статичный спрайт рыбака в соломенной шляпе на лодке.
2. **Заброс:** По нажатию `Space` в воде появляется объект поплавка.
3. **Ожидание:** Через случайный промежуток времени срабатывает триггер поклевки (анимация поплавка).
4. **Улов:** Повторное нажатие `Space` во время поклевки выводит на экран карточку с изображением рыбы и её параметрами (вид, вес).

## Как запустить
1. Склонировать репозиторий.
2. Открыть файл решения `.sln` в Visual Studio.
3. Установить зависимости через **NuGet** (пакеты `sdl2`, `sdl2.image`, `sdl2.ttf`).
4. Выполнить Build и запустить проект.

## Зависимости
* **SDL2 Core** — управление окном и вводом.
* **SDL_image** — рендеринг персонажа, фона и пойманной рыбы из PNG.
* **SDL_ttf** — отрисовка текста с характеристиками улова.

* **Space** — Заброс удочки / Подсечка.
* **Esc** — Выход из игры.
