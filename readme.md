# CalmWaters

Реализация расслабляющего 2D-симулятора рыбалки на языке C++ с использованием графической библиотеки SDL3 и системы сборки CMake.

## Описание

**CalmWaters** — это минималистичная аркадная игра про рыбалку. Игрок управляет рыбаком на лодке, перемещаясь по водоёму, выбирая идеальное место для заброса и соревнуясь в улове. 

**Основные механики:**
* **Управление лодкой:** Перемещение влево и вправо по водоёму для поиска рыбных мест.
* **Заброс удочки и поплавка:** Динамический заброс в точку клика мыши с физикой полёта.
* **Мини-игра «Натяжение лески»:** При поклёвке запускается фаза вываживания, где нужно удерживать баланс натяжения лески клавишей `Пробел`, чтобы рыба не сорвалась и леска не порвалась(В разработке!).
* **Процедурный мир рыб:** Случайная генерация улова (Окунь, Карась, Карп, Лещ, Щука, Сом) со случайным весом, временем ожидания поклёвки и категорией редкости (обычная, редкая, легендарная).
* **Внутриигровой магазин:** Возможность открывать и покупать новые визуальные скины для лодок за заработанные очки улова.
* **Ретро-атмосфера:** Встроенный процедурный синтезатор звуковых волн для генерации фоновой музыки в реальном времени.

## Требования

Для успешной компиляции и запуска проекта в вашей системе должны быть установлены:

* Компилятор с поддержкой стандарта **C++17** или выше (MSVC 2019+, GCC 11+ или Clang)
* **CMake** версии 3.8 или выше
* Установленная в системе библиотека **SDL3** (пути по умолчанию настроены для Windows в `C:/SDL3/SDL3-3.4.8`)
